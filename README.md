## Supermarket Customer Behaviour & Revenue Analysis

Analysis of supermarket transaction data using Probability and Statistics in R.

## Files
- `Proj new.R` — R Studio code for statistical analysis
- `SuperMarket Analysis.csv` — Dataset of 1,000 transactions
- `supermarket-customer-behaviour-probability-revenue-analysis.pptx` — Presentation

## Topics Covered
- Measures of Central Tendency & Dispersion
- Probability Distributions (Binomial, Poisson, Normal)
- Bayes Theorem
- Frequency Tables & Histograms
- Central Limit Theorem

## Dataset
- 1,000 transactions across 3 branches (Alex, Giza, Cairo)
- 6 product lines
- Jan–Mar 2019
